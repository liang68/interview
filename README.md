## 1. 计算一个集合所有的子集
+  需要认识的：空集是任何集合的子集；真子集为不包含子集的集合；非空真子集即不包含子集与空集合
+  这道题可以使用**“按位对应法”**进行计算
+  如集合A={a,b,c},对于任意一个元素，在每个子集中，要么存在，要么不存在。 映射为子集：<br />
 (a,b,c) <br />
 (1,1,1)->(a,b,c) <br />
 (1,1,0)->(a,b) <br />
 (1,0,1)->(a,c) <br />
 (1,0,0)->(a) <br />
 (0,1,1)->(b,c) <br />
 (0,1,0)->(b) <br />
 (0,0,1)->(c) <br />
 (0,0,0)->@(@表示空集)<br />
 观察以上规律，与计算机中数据存储方式相似，故可以通过一个整型数与集合映射...000 ~ 111...111（表示有，表示无，反之亦可），通过该整型数逐次增可遍历获取所有的数，即获取集合的相应子集。
